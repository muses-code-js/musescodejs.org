What it's all about:

This is a study group event that will happen on a regular basis for 3 months (March-April-May, then we will continue in the year in September-October-November), and the amount of attendees is very limited, so, please, RSVP if you come for sure! and let us know if you are not able to attend at least 3 days prior to the date. The topics and focus points for each session would be announced prior to the event via email, so keep an eye on your emails and add info@musescodejs.org to your trusted emails as sometimes they get into the spam folder. The reason why we hold it for a small group is to give an opportunity to everyone to participate on maximum. We also divide groups into 2-3 smaller groups based on your level of programming skills.

Our goal is to learn, understand and improve our technical practices and communication skills with fellow software developers in a non-competitive and inclusive setting and promote continuous learning.

For the session we will have few 'Katas' - so-called exercise - on which we will work during the evening, we will be learning HTML & CSS. HTML is the foundation behind all web pages. It’s used to add structure and form to text, images, and more. CSS is the language used to style HTML content. In this first course, you’ll learn the fundamentals of both languages so that you can create visually appealing web pages. Each session is an independent session but if when we have some people returning from session to session we have a bigger challenge for them which they continue working on.

What you'll learn:

  • - What is HTML and CSS?
  • - Basic terminology and syntax
  • - HTML tags
  • - File naming and project setup
  • - Anchors and attributes
  • - Lists and Images
  • - Intro to CSS
  • - Styling your page
  • - Classes and Ids

If you have any lightening talks to share, please let the organisers know beforehand, simply email date of event and light talk topic to info@musescodejs.org
